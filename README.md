# Titanic Dataset Analysis

## Project Overview
This project analyzes the Titanic dataset to uncover patterns and trends that influenced passenger survival during the Titanic disaster. The analysis uses statistical summaries, visualizations, and correlation analysis to identify key factors affecting survival.

---

## Objectives
- Explore and understand the Titanic dataset
- Perform data cleaning and preprocessing
- Generate statistical summaries
- Create visualizations to identify patterns
- Analyze correlations between variables
- Identify key factors influencing survival

---

## Technologies Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

---

## Dataset Information
The dataset contains information about Titanic passengers, including:
- Passenger class
- Gender
- Age
- Fare
- Survival status
- Family information

Dataset Size:
- Rows: 891
- Columns: 12

---

## Data Cleaning
The following preprocessing steps were performed:
- Checked for missing values
- Removed the Cabin column due to excessive null values
- Verified dataset structure and data types

---

## Analysis Performed
### Statistical Analysis
- Mean, median, minimum, and maximum values
- Category counts
- Dataset summary statistics

### Visualizations
- Survival Distribution
- Gender vs Survival
- Passenger Class vs Survival
- Age Distribution
- Fare Distribution
- Correlation Matrix

### Correlation Analysis
Relationships between numerical variables were analyzed to identify factors affecting survival.

---

## Key Insights
- Female passengers had higher survival rates than male passengers.
- First-class passengers survived more frequently than third-class passengers.
- Higher ticket fares were associated with increased survival chances.
- Passenger class and gender were the strongest influencing factors.

---

## Conclusion
The analysis revealed that survival on the Titanic was strongly influenced by gender, passenger class, fare, and age. Statistical summaries and visualizations helped uncover meaningful trends and relationships within the dataset.
